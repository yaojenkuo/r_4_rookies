## R 語言新手村

### 簡介

這是[郭耀仁](https://www.facebook.com/yaojen.kuo.1)在[台灣大學資訊工程學系資訊系統訓練班](https://www.csie.ntu.edu.tw/train/)所開設的 R 語言新手村課程網頁，課程時數 10 小時。

### 評分標準

- 30% 出席率
- 50% 期中作業
- 20% 期末作業
